# Stack

- Astro static output for a low-maintenance, serverless portfolio.
- Tailwind CSS with project-owned CSS tokens for responsive styling and theming.
- pnpm 10.23.0 and Node 22.21.1, pinned for reproducible local and CI builds.
- Astro asset processing for responsive gallery images.

The site deliberately has no backend. Browser-only enhancements must remain optional and must not block the static page from rendering.
