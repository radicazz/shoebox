# shoe-box

Lightweight personal Pinterest-style portfolio gallery.

## quick-start

```bash
# Grab the code
git clone https://www.github.com/radicazz/shoe-box.git/
cd shoe-box

npm install

npm run dev      # Local dev server at http://localhost:4321
npm run build    # Production build to dist/
```
